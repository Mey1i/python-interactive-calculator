# Python Interactive Calculator

A simple yet powerful interactive calculator written in Python.  
It supports addition, subtraction, multiplication, and division using a clean function-mapping system.  
Users can continue calculations with the previous result, start a new session, or exit at any time.

1. The user enters the first number.

   - Restart with a new number
   - Quit the program
